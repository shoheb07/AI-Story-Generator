# AI-Story-Generator
The AI Story Generator is a web-based application developed using HTML, CSS, and JavaScript. The application generates creative stories automatically based on user-provided topics.

# AI Story Generator

## Project Overview

The AI Story Generator creates random creative stories based on user topics using JavaScript.

The project demonstrates dynamic text generation and frontend web development concepts.

# Features

- Topic-based story generation
- Interactive interface
- Random story templates
- Responsive design


# Technologies Used

- HTML5
- CSS3
- JavaScript
